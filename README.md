## 🧠 Core Philosophy: The Red Car Theory

Our protocol is built on the Reticular Activating System (RAS) of the human brain.

1.  **The Prime:** We show a generic category icon (e.g., "Find the Shoe") *before* the ad to create a "knowledge gap".
2.  **The Search:** The user watches the video actively to identify the specific model.
3.  **The Match:** The user verifies their finding in a rapid post-ad challenge.

## 🎮 UX Architecture (v9.0)

### 1. The Mystery Primer
No spoilers. Just a category prompt to ensure the user watches the content.

### 2. The Safe Zone (Video)
* **No-Click Policy:** Clicking the video player does nothing. We respect the gamer.
* **Ghost Button:** A non-intrusive "Install" button fades in after 3 seconds for high-intent users.

### 3. Eagle Eye Challenge
A simple 3-card grid. The user must select the product shown in the video.
* **Smart Hints:** If the user hesitates (>5s), wrong options fade out automatically.
* **Super Hint:** After 8s, the correct option pulses to guide confused users (Accessibility).
* **Hybrid Input:** Users can **Hold** or **Rapid Tap** to confirm.

### 4. Action Hub
* **Install Now:** Direct link to store.
* **Save for Later:** Uses the Native System Share Sheet (Zero permissions required).

## 💰 Unit Economics: The "Hybrid Floor"

We eliminate financial risk for publishers via a dynamic tiering engine:

| Tier | User Behavior | Revenue Model |
| :--- | :--- | :--- |
